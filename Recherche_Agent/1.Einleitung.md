🧠 eduScrum-Recherche-Agent – Setup für deine IT-Umschulung

Mit diesem Agenten suchst du ein Thema, bekommst die Kerninfos und erhältst eine fertige eduScrum-Lernkarte (Essenz, Lernziele, Akzeptanzkriterien, Merksätze, Analogie, Quellen). Alles läuft lokal, mit freier Software – keine API-Kosten, keine Cloud-Abhängigkeit.

Zeitaufwand: ca. 4–6 Stunden inkl. erster Tests – perfekt für 2 Tage Urlaub.
🎯 Was du hinterher hast

    Eine eigene lokale Suchmaschine (SearXNG)

    Ein leistungsfähiges Sprachmodell (Ollama + Llama 3.1 8B)

    Den GPT-Researcher-Agent, der autonom recherchiert und zusammenfasst

    Ein Prompt-Template, das die Ergebnisse in dein eduScrum-Format gießt

📋 Voraussetzungen

Bevor du startest, stelle sicher, dass dein Rechner folgendes erfüllt:
Was	Empfehlung	Notwendig?
Betriebssystem	Windows 10/11, Ubuntu 22.04+, macOS 13+	Ja
RAM	min. 16 GB, besser 32 GB	Ja (Ollama braucht Speicher)
Freier Festplattenplatz	ca. 20 GB (Modelle + Docker-Images)	Ja
Git	git-scm.com	Ja (um Repo zu klonen)
Docker Desktop	docker.com	Ja (für SearXNG)
Python 3.10+	python.org	Ja (GPT Researcher)
Grundkenntnisse Terminal	Du kannst cd, git clone, pip install ausführen	Hilfreich

🔹 Falls etwas fehlt, lies den jeweiligen Abschnitt in der Anleitung – ich gebe dir für jedes Tool die nötigen Kurz-Installationshinweise.
