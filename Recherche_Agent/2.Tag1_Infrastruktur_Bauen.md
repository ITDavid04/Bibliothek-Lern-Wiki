🗓️ Tag 1 – Infrastruktur bauen (ca. 2-3 h)
1. Docker und SearXNG einrichten (30 min)

Warum: SearXNG ist deine private Metasuchmaschine, die der Agent für die Websuche nutzt – keine Google API, kein Tracking.
1.1 Docker Desktop installieren

    Lade dir Docker Desktop herunter und installiere es.

    Nach der Installation starte Docker Desktop und warte, bis es bereit ist (grünes Symbol).

1.2 SearXNG als Container starten

Öffne ein Terminal (PowerShell, CMD oder Bash) und führe aus:
bash

docker run -d --name searxng -p 8080:8080 searxng/searxng

Test: Öffne den Browser unter http://localhost:8080. Du solltest eine Suchmaske sehen. Gib einen Testbegriff ein – wenn Ergebnisse kommen, läuft SearXNG.

    💡 Warum Port 8080? Das ist der Standard. Sollte der Port belegt sein, ändere -p 9090:8080 und merke dir den Port.

Docker-Stopp und Neustart:

    Stoppen: docker stop searxng

    Wieder starten: docker start searxng

    Nach dem Reboot läuft der Container nicht automatisch – starte ihn bei Bedarf neu.

2. Ollama und dein erstes lokales LLM (45 min)

Warum: Ollama verwaltet lokale KI-Modelle und stellt eine API bereit, die der Agent anspricht. Keine Cloud, keine Kosten.
2.1 Ollama installieren

Gehe auf ollama.com und lade den Installer für dein Betriebssystem herunter. Führe die Installation aus.
2.2 Dein Modell herunterladen

Öffne ein Terminal und gib ein:
bash

ollama pull llama3.1:8b

Das Modell ist etwa 4,7 GB groß – der Download kann ein paar Minuten dauern.
Danach kannst du es testen:
bash

ollama run llama3.1:8b

Es erscheint ein Prompt, du kannst z.B. tippen: "Erkläre Docker in zwei Sätzen". Beenden mit /bye.

Wichtig: Das Modell benötigt RAM – mit 16 GB läuft es, aber schließe andere speicherhungrige Programme.

Alternatives Modell (falls du weniger RAM hast): ollama pull phi3:mini (2,3 GB)

🔧 Port-Check: Die Ollama-API läuft standardmäßig auf http://localhost:11434. Das brauchst du später.
3. Python-Umgebung für GPT Researcher vorbereiten (30 min)
3.1 Python installieren (falls nicht vorhanden)

Lade Python von python.org herunter. Wichtig: Beim Installieren „Add Python to PATH“ anhaken.
3.2 Git installieren (falls nicht vorhanden)

Lade es von git-scm.com und installiere es mit Standardeinstellungen.
3.3 Projektverzeichnis anlegen und Repo klonen

Öffne ein Terminal und wechsle in deinen Wunschordner, z.B. C:\Projekte (Windows) oder ~/projekte (Linux/macOS):
bash

cd ~/projekte
git clone https://github.com/assafelovic/gpt-researcher.git
cd gpt-researcher

3.4 Virtuelle Umgebung und Abhängigkeiten

Erstelle eine virtuelle Umgebung (empfohlen) und installiere alles Notwendige:
bash

python -m venv venv

# Windows
venv\Scripts\activate

# macOS/Linux
source venv/bin/activate

pip install -r requirements.txt

Das installiert alle Pakete, die der Agent braucht.
